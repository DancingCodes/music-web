## 专注听歌 / 歌词页

点击播放栏的 🎤 或封面图进入，全屏铺满，背景为封面图模糊 + 暗色渐变遮罩。

```
┌──────────────────────────────────────────────────┐
│  封面渐隐背景（占据整个页面）                      │
│                                                  │
│              ┌────────┐                          │
│              │ 🖼     │   ← 大封面（旋转中）      │
│              └────────┘                          │
│                                                  │
│              七里香                               │
│              周杰伦                               │
│                                                  │

│  │         00:37.24 作曲 : 周杰伦  ← 当前行高亮  ││
│  │         00:42.18 窗外的麻雀                   ││

│  01:15 ──────────●─────────── 05:02              │
```

- 歌词区域居中，当前行高亮（主题色 + 字号变大），其他行半透明
- Howler.js 实时回调 `currentTime` 匹配 `[mm:ss.xx]` 时间戳，自动滚动到当前行
- 封面可旋转动画（播放时旋转，暂停时停止）
- 点击任意歌词行跳转到对应时间
- 背景封面图 10px 高斯模糊 + 半透明深色蒙层

## 主题切换

- 导航栏右侧 ☀️/🌙 按钮一键切换
- Tailwind `dark:` 实现，全局 `class="dark"` 在 `<html>` 切换
- 暗色：深灰底（`slate-900`）+ 白字，卡片用 `slate-800`
- 亮色：白底 + 黑字，卡片用 `gray-50` 边框
- 歌词页背景始终为封面模糊 + 蒙层，不受主题影响
- 主题偏好存 `localStorage`

## 交互

- **搜索**：回车触发，结果卡片展示；保存按钮 loading → 成功变「已收藏」
- **曲库筛选**：输入自动请求 `?name=` 筛选
- **删除**：确认弹窗「确定删除 XXX？同时会删除云端文件」
- **播放栏**：固定底栏，进度条可拖拽，音量滑条，🎤 进歌词页
- **歌词页**：全屏遮罩，封面旋转 + 歌词自动滚动，当前行高亮，点击歌词跳转，点击背景或「收起」返回
- **空状态**：曲库无数据时显示图标 + 「还没有收藏歌曲，去搜索吧」
- **Toast**：右上角弹出，成功绿色 ✓，失败红色 ✕，2.5s 消失
